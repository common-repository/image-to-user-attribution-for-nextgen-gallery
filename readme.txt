== Description ==

**The plugin saves the image's author (uploader) name along with the uploaded image information.**     
*The "Image-to-user attribution for Nextgen Gallery" plugin is useful if you are building a public gallery       
where people can bring their own contribution.*    

== Installation ==

1. Extract the zip file.
1. Upload the folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
